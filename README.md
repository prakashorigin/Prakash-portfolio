# Prakash-portfolio

A modern developer portfolio website built using React and TypeScript.

## Features

- Projects showcase
- Skills section
- Resume/CV
- Contact form
- Responsive design
- Dark mode theme

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js
- **Deployment**: Vercel

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to view the portfolio.

## Build

```bash
npm run build
```
